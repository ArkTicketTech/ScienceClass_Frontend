## 动态表单 Web 前端

## 配置 
```
$ npm install               # Install the dependencies
$ npm install -g webpack    # Install webpack
```

## 运行

```
$ npm run start
```

会自动监听 3001 端口，在浏览器中输入 localhost:3001 即可看到效果

## 前端资源管理
```
$ npm run watch
```

会自动监听前端资源(.css, .js 文件)的更改并自动 build

## 说明

- css: 原始 less 文件
- js: 原始 js 文件
- dist: 经过 webpack 合成的 bundle.js 文件
- lib: 第三方库
